# CheckmyGrade App 

This is my version of CheckmyGrade App for DATA 200 Lab 1. 

## Main Features:

1. Secure Login System using SHA-256 password encryption  
2. Student Management – Add, delete, update, search, sort student records  
3. Professor Management – Add, delete, update professors  
4. Course Management – Manage course details (add/delete/display) 
5. Grades System – Automatic grade assignment based on marks
6. Reports and Statistics – Calculate average and median marks  
7. Performance Measurement – Displays time taken for search and sort operations  
8. Unit Testing – Automated test coverage for CRUD operations using **unittest**

## Running code

`python CheckMyGrade_lab_work_1.py`
